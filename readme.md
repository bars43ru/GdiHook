Иследования перехвата отрисовки текста в окне стороннего процесса.

Для корректной работы приложения необходимо зарегестрировать следующи dll

```
regsvr32.exe /s "<путь к директории>\DeviareCOM.dll"
regsvr32.exe /s "<путь к директории>\DeviareCOM64.dll"
```

 Где `<путь к директории>` полный путь к директории с файлами DeviareCOM*.dll (в проекте директория `.\src\Resources\Deviare2\*`)


Снятие регистрации:

```
regsvr32.exe /u "<путь к директории>\DeviareCOM.dll"
regsvr32.exe /u "<путь к директории>\DeviareCOM64.dll"
```
